Detailed Description

Student database.

Contains the details of the Students or the Professors. Every row in database is unique to a particular user. This is confirmed by unique username and password.

Note
isProfessor is a Boolean field. If checked, it means the corresponding User gets the priveleges of a professor. This can only be set by the django-admin.
